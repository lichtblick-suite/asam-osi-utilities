# Developer Workflow

This page contains day-to-day commands. For environment setup, see [Development Setup](setup.md).

## Update Repository

```bash
git pull --recurse-submodules
# If needed:
git submodule update --init --recursive
```

## Build and Test

**vcpkg presets:**

```bash
cmake --preset vcpkg-linux    # Linux
cmake --preset vcpkg-windows  # Windows
cmake --preset vcpkg          # macOS
cmake --build build -j
ctest --test-dir build --output-on-failure
```

## Format and Lint

Use the hooks generated by the setup script:

```bash
.git/hooks/pre-commit
.git/hooks/pre-commit --fix
```

Notes:

- Lint runs automatically inside the hook when `clang-tidy` is available.
- It requires `build/compile_commands.json` (generated by the build).

## Commit

```bash
git commit -S -s -m "feat: short description"
```
